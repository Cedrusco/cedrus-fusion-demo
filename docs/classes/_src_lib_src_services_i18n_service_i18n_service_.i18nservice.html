  • formatDateWithTokens(value: any, format: string): string
    • Defined in src/lib/src/services/i18n-service/i18n.service.ts:86

    Format date according to the current locale.

    Date format corresponds to the moment.js tokens:

    LT - Time

    LTS - Time with seconds

    L - Month numeral, day of month, year

    l - Month numeral, day of month, year - shortcut

    LL - Month name, day of month, year

    ll - Month name, day of month, year - shortcut

    LLL - Month name, day of month, year, time

    lll - Month name, day of month, year, time - shortcut

    LLLL - Month name, day of month, day of week, year, time

    llll - Month name, day of month, day of week, year, time - shortcut

setComponentDictionaryURL

  • setComponentDictionaryURL(path: string, componentName: string): Promise<any>
    • Defined in src/lib/src/services/i18n-service/i18n.service.ts:122

    Sets new dictionary URL for the particular component.

    Dictionary files must be named according to the locales, ex: en.json, fr.json etc.

    Therefore, if the URL is "http://my_site/my_dictionary", the system will try to find new dictionaries under "http://my_site/my_dictionary/en.json", "http://my_site/my_dictionary/fr.json" etc.

    NOTE: If dictionary for this path and locale is already in memory, then it will NOT be reset.

    Parameters

    • path: string
    • componentName: string

    Returns Promise<any>
