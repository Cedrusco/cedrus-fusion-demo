cfDatatable

cfDatatable: DatatableModel
  • Defined in out-tsc/lib/src/components/datatable/datatable.component.ts:72

It is Object which will recive data from DatatableModel Class. It is used in cf-datatable template.

Example of DatatableModel Class:

{
        rows: [
            { name: 'Austin', city: 'London' },
            { name: 'Molly', city: 'Tokyo' },
            { name: 'Austin', city: 'New York' },
            { name: 'Monica', city: 'Paris' },
            { name: 'Austin', city: 'Dubai' }
        ],
        columns: [
            { name: 'Name', type: 'standard' },
            { name: 'City', type: 'standard' }
        ],
        rowHeight: 50,
        limit: 3,
        filtering: true,
        expandable: {
            value: true,
            detailsHeight: 140
        },
        selected: this.mySelected
}

rows: array

It is an array with data for rows of any type.

columns: array

It is an array with columns definition. Each column item must have two required properties: name and type. Name will be used as column header. Type will set layout of each column and can be: standard(default), cf-icon, cf-button.

rowHeight: number

It is height of each row. Can be numeric (number will be transformed to pixels) or 'auto'. Optional. Not set by default. Required when rows expanding is set.

limit: number

Means rows quontity per page. Optional.

filtering: boolean

It means if Filter must be builded to Datatable. Type boolean. Optional. Default false.

expandable: object

It is object with properties for rows details. Optional.

value property of expandable: boolean

It is boolean value to build or not row details.

detailsHeight property of expandable: boolean

It is number value and mean height of row in pixels. Required.

selected: array

Array for selected items. Optional.
