cfToolbar

cfToolbar: ToolbarModel
  • Defined in src/lib/src/components/toolbar/toolbar.component.ts:80

It is an attribute [properties] of directive cf-toolbar which take ToolbarModel object.

Example of model:

{
 template: 'my-toolbar-template-class',
    info: {
        show: true,
        icon: this.myInfoIcon,
        order: 'first-section'
    },
    content: {
        title: 'This is my toolbar title',
        order: 'second-section'
    },
    actions: {
        show: true,
        menu: this.myToobarMenu,
        order: 'third-section'
    },
    close: {
        show: true,
        icon: this.myCloseIcon,
        order: 'fourth-section',
    },
    toggle: {
        show: true,
        minimizeIcon: this.myMinIcon,
        maximizeIcon: this.myMinIcon,
        order: 'fifth-section'
    },
    collapsible: {
        show: true,
        expanded: true,
        icon: this.myCollapsibleIcon
        order: 'fifth-section'
    }
}

template: string

Name of css class with styles for elements

info, content, actions, close, toggle, collapsible: object

Objects with data for toolbar sections

show: boolean

Property to show / hide specific toolbar section

order: string

Order of toolbar section. Can be: first-section, second-section, third-section, fourth-section, fifth-section, sixth-section

expanded: boolean

Toolbar collapsible section property to be used on some html element to show / hide it content
