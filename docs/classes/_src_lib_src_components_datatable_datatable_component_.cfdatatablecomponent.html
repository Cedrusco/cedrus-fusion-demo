onSort

onSort: EventEmitter<any> = new EventEmitter<any>()
  • Defined in src/lib/src/components/datatable/datatable.component.ts:257

Output event with four properties: column, newValue, prevValue, sorts.

Column is data with ngx-datatable column properties.

NewValue can be: asc(ascending sorting), desc(descending sorting) or undefined(not sorted).

PrevValue can be: asc(ascending sorting), desc(descending sorting) or undefined(not sorted).

Sorts is an array with sorting object(s).

properties: DatatableModel

    selectable: boolean,
    sorted: [],
    selected: any[],
}

rows: array

It is an array with data for rows of any type.

rowHeight: number

It is height of each row. Can be numeric (number will be transformed to pixels) or 'auto'. Default: 50. Required when rows expandable is set.

limit: number

Means rows quontity per page. Default: undefined.

filterable: boolean

It means if filtering input must be shown. Type boolean. Default: false.

filterProperty: InputModel

CfInput component for the filter.

expandable: boolean

It means if to build or not row details. Default: false.

expandingIconProperty: IconModel

CfIcon component for the expanding icon.

collapsingIconProperty: IconModel

CfIcon component for the collapsing icon.

detailsHeight: boolean

It is number value and mean details row height in pixels. Default: 130.

selectable: boolean

It means if to show or not row selection checkboxes. Default: false.

sorted: array

Array with sorting objects. Default: []. Rows can sorted by multipe columns. Each sorting object is a json object with two properties: prop(property name by which to sort rows) and dir(sort direction. Can be: asc or desc).

selected: array

Array for selected items.

Example:

{
        rows: [
            { name: 'Austin', city: 'London' },
            { name: 'Molly', city: 'Tokyo' },
            { name: 'Austin', city: 'New York' },
            { name: 'Monica', city: 'Paris' },
            { name: 'Austin', city: 'Dubai' }
        ],
        rowHeight: 30,
        limit: 3,
        filterable: true,
        filterProperty: this.myInput
        expandable: true,
        expandingIconProperty: this.myExpandingIcon
        collapsingIconProperty: this.myCollapsingIcon
        detailsHeight: 150
        selectable: true,
        sorted: [{prop: 'name', dir: 'desc'}],
        selected: this.mySelected
}

        dynamicClass,
        class
    },
    // Top section for filtering/expanding
    topOptions: {
        dynamicClass,
        class
    },
    // Filter element
    inputFilter: InputStylingModel, //refer to input component
    // Expanding icon styling
    expandingIcon: IconStylingModel, //refer to icon component
    // Collapsing icon styling
    collapsingIcon: IconStylingModel, //refer to icon component
    // Ngx-datatable elelemnt
    table: {
        dynamicClass,
        class
    },
    // Each cell container
    tableCell: {
        dynamicClass,
        class
    },
    // Each row details section
    tableRowDetails: {
        dynamicClass,
        class
    }
}

Example:

    new DatatableStylingModel({
        topOptions: {class: 'my-class-for-top-options'},
        inputFilter: new InputStylingModel({
            iconStyling: new IconStylingModel({ icon: { themeColor: "primary" } })
        }),
        expandingIcon: new IconStylingModel({ icon: {themeColor: "primary", class: "rotated"} }),
        collapsingIcon: new IconStylingModel({ icon: {themeColor: "primary", class: "rotated"} }),
        table: {class: 'my-class-for-table'},
        tableCell: {class: 'my-class-for-table-cell'},
        tableRowDetails: {class: 'my-class-for-table-row-details'},
