Class CfTreeviewComponent

Treeview component based on Angular 2 Tree. Treeview items has options to be selectable, expandable, dragable and has templating possibility for aech item.

Example of using Menu:

<cf-treeview [properties]="myTreeview" [styling]="myTreeviewStyles"></cf-treeview>

properties

properties: TreeviewModel
  • Defined in out-tsc/lib/src/components/treeview/treeview.component.ts:67

It is an attribute [properties] of directive cf-treeview which take TreeviewModel object.

Example of Treeview json object:

{
 items: [
   { _id: '1', _name: 'root1', _expanded: true, _children: [
     { _id: '2', _expanded: false, _name: 'child1' },
     { _id: '3', _expanded: false,_name: 'child2' }]
   }
 ],
 options: {
   idField: '_id',
   nameField: '_name',
   childrenField: '_children',
   isExpandedField: '_expanded',
   selectable: true,
   allowDrag: true
 }
}

items: array

Ii is an array of tree items. Each item must have two required properties: id with unique id and name.

options: object

It is an object with specific properties. All properties are not required.

allowDrag: boolean

Means items to be dragable. Default is false.

selectable: boolean

Means items to be selectable by checking checkboxes. Default is false.

idField: string

Means name of property to be mapped as id. If not set - component will be looking property id inside item.

displayField: string

Means name of property to be mapped as name. If not set - component will be looking property name inside item.

childrenField: string

Means name of property to be mapped as children. If not set - component will be looking property children inside item.

isExpandedField: string

Means name of property to be mapped as expanded. If not set - component will add property expanded inside item.
