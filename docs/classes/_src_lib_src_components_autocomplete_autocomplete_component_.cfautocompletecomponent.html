properties

properties: AutocompleteModel

Overrides CfCoreComponent.properties

  • Defined in src/lib/src/components/autocomplete/autocomplete.component.ts:72

AutocompleteModel object.

Example of model:

{
 displayField: "label",
 source: this.mySource,
 useChips: true,
 chipsSelected: "multiple",
 chipsReadOnly: false,
 multiselection: false,
 uniqueSelected: true,
 stayOpenedOnClick: true,
 highlighting: false,
 prefetch: true,
 cache: true,
 delay: 1000
}

List of all model properties

displayField: string

Name of item property to be used as item label and selected value. Required.

source: any[]

Array with items. Optional. Has lower priority then 'serverUrl'.

serverUrl: string

Server url to be called for to take items. Optional. Has highter priority then 'source'.

useChips: boolean

Using chips for selected items. Optional. Default: false.

chipsSelected: string

If useChips is true, then it is possible to set chips selection: 'single' or 'multiple'. Optional. Default: 'multiple'.

chipsReadOnly: boolean

If chips can't be removed after selection. Optional. Default: false.

multiselection: boolean

If dropdown item can be selected multiple times. Optional. Default: true.

uniqueSelected: boolean

Search items with using comma inside search input field. Optional. Default: false.

stayOpenedOnClick: boolean

If dropdown must be opened on selecting items. Optional. Default: false.

highlighting: boolean

If to highlight search value in items labels. Optional. Default: false.

cache: boolean

If to cache search results. Optional. Default: false.

prefetch: boolean

If to send request to a server on component init. Optional. Default: false.

delay: boolean

The amount of time in milliseconds from when the user stops typing to when the result is returned. Optional. Default: 250.
