properties

properties: GalleryModel
  • Defined in out-tsc/lib/src/components/gallery/gallery.component.ts:79

It is an attribute [properties] of cf-gallery which take GalleryModel object.

Example of GalleryModel object:

{
 options: [
   {
     width: '100%',
     imagePercent: 60,
     imageArrowsAutoHide: true,
     thumbnailsPercent: 40,
     thumbnailsRows: 2,
     thumbnailMargin: 1,
     thumbnailsMargin: 1,
     previewFullscreen: true
   }
 ],
 images: [
   {
     small: 'https://i1.wallpaperscraft.com/image/nature_waterfall_summer_lake_trees_90400_300x168.jpg',
     medium: 'https://i1.wallpaperscraft.com/image/nature_waterfall_summer_lake_trees_90400_602x339.jpg',
     big: 'https://wallpaperscraft.com/image/nature_waterfall_summer_lake_trees_90400_1920x1080.jpg',
     description: 'Nature waterfall summer lake_trees'
   },
   ...
 ]
 }

options: array

It is an array with options objects type NgxGalleryOptions. It is an array because of 's for responsive styling. For each breakpoint it is possible to define different gallery styles.

width: string

It is width of galley in pixels or percentage. Optional. Default is '500px'.

height: string

It is height of galley in pixels or percentage. Optional. Default is '400px'.

imagePercent: number

It is height of main image according to gallery height. Optional. Default is 75.

imageArrowsAutoHide: boolean

It is auto hide of image left/right arrows. Optional. Default is false.

thumbnailsPercent: number

It is height of thumbnails container according to gallery height. Optional. Default is 25.

thumbnailsRows: number

It is quontity of thumbnails rows. Optional. Default is 1.

thumbnailMargin: number

It is margin between images in thumbnails. Optional. Default is 10.

thumbnailsMargin: number

It is margin between thumbnails and image. Optional. Default is 10.

previewFullscreen: boolean

It is option to have fullscreen in preview. Optional. Default is false.

images: array

It is array with images. Each image is an object with such properties: small, medium, big, description.

small: string

It is image url for thumbnail.

medium: string

It is image url for main gallery image.

big: string

It is image url for opened image.

description: string

It is title for image.
