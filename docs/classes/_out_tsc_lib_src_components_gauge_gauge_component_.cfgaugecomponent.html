Class CfGaugeComponent

Gauge component is a modification of Angular gauge with added label, prefix and suffix elements. All elements can be customized and stylized.

Example of using:

<cf-gauge [properties]="myGauge" [styling]="myGaugeStyles"></cf-gauge>
