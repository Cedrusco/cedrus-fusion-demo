properties

properties: MenuModel
  • Defined in out-tsc/lib/src/components/menu/menu.component.ts:60

It is an attribute [settings] of directive cf-menu which take array with menu items.

Example of menu item json object:

{
    label: 'My menu item label',
    type: 'standard',
    icon: Icon,
    notification: '5',
    display: true,
    disable: false,
    onClick: this.myFunction
}

label: string

It is label of menu item. Default is Menu item label.

type: string

It is menu item type. Types are: standard and divider. Default is standard.

icon: Object

It is menu item icon generated by Icon Class (look to Icon API Referance).

notification: string

It is menu item notification. Can mean number of notifications.

display: Boolean

It is menu item visibility. Default is true.

disable: Boolean

It is menu item disable status. Default is false.

onClick: any

It is menu item click event.
