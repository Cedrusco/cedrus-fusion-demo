Template System

Fusion Components can be configured using the templating system

Setup

1- Download this folder

2- Add the folder to your project

3- On the root of your project add a : fusion-config.json with the following structure: 

        * required
      
        {{ '{' }}
          * templateUrl:string       //url to the templates folder downloaded and added to your project,
            defaultDirectory:string  //default templates folder found in the templates directory
        {{ '}' }}

Custom Template

To create your own custom template you need to:

1- Copy/Paste Default folder under Template directory

2- Rename new folder to your own custom templates name

3- Edit the component templates

If you have one file for a custom component then the name of the file should remain same as found in the default directory

If you have more than one custom template for the same component, one of them should name same as the one found in the default, and the rest can be named to your preference

By default if only the directory is specified in the compTemplate of the component, then the file named as the one found in the default directory will be applied

EXAMPLE

1- Copy/Paste Default directory and rename to 'MyTemplates'

2- Change button-template.json to my own custom template

3- Create another template for the button and name it myCustom-button-template.json

To apply button-template.json you can either specify it explicitly like this: compTemplate="MyTemplates/button-template.json" or compTemplate="MyTemplates" which will automatically apple the default naming of the component

To apply myCustom-button-template.json you have to explicitly do so like this: compTemplate="MyTemplates/myCustom-button-template.json"

Apply on Component

To apply a template on a component all you need to do is

      
      <cf-component compTemplate=“folderName”></cf-component>

Folder name will be the directory under which your custom component templates are found

By default if no template is defined then the default is applied

If you specify a default directory in the fusion-config.json file then that directory will be your default one

If you don't specify a compTemplate on a component, then the default is applied (fusion default, or your own if you specify it in the config)

More specifically the default directory and the name of the template following this convention: component-template.json
