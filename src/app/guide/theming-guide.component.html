Theming and Styling

Theming

Most of the fusion components are build on top of Angular Material components

So to apply a theme on the fusion components follow the guide of theming angular material component found Here

To specify a theme on a fusion component you can either pass the theme color as an input to the component

<cf-component themeColor="primary or accent or warn"></cf-component>

Or you can specify it in the component's styling model explained below

Remark: If a fusion component is not built on top of an angular material component, then the theming is not supported for the moment

Styling

To style a fusion component, the component's styling model should a passed to the component as an input

<cf-component styling="component styling model"></cf-component>

Refer to the component needed to know what the styling model is

For every fusion component, you will under it's styling model a container attribute.This will refer to the 'div' surrounding that component

For every attribute in the styling model, you will find 3 ways to style it:

1- dynamicClass: User needs to pass a function that returns the name of the css class depending on a certain criteria.

Example: If I have a cart button that you press on to see what is in your shopping cart, then by default this button I want it to look different then when the cart items are more than one.

In that case, I will have a checkCart() function that depending on the cart count I will return a 'fullCart' or 'emptyCart' class found in my css file

2- class: Name of the css class to apply on the component

3- themeColor: primary/accent/warn Remark: Only supported on components built on top of angular material
