Demo features:

  • Used sub-components: header (with pager inside), footer (with filter and rows-expander inside)
  • Datatable rowExpand property is set to single (only one row can be expanded at the same time)
  • Details template is provided
  • Columns goes with their headerId / contentId and their templates
  • Columns has diferent type width and datatable has property columnMode === 'standard'
  • All columns has diferent header / rows cells alignments
  • Datatable rows expand/collapse icons are customized
  • Datatable does with some styling model
