The default directory is set in fusion-config.json along with the path to the templates folder.

  1. By default, the component will take component-template.json. from the default directory under templates folder.

  2. To specify a new directory, set compTemplate property to the name of that directory, and the component will take it's relative template component-template.json.

  3. If multiple templates for the same component under 1 directory, then set compTemplate to the path of your template with the name of the template.
