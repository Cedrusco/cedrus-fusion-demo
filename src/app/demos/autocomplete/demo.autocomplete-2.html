Demo features:

  • Source data is taken from server temp-site
  • prefetch is true, so on component init external service call will be sent to take results and cached
  • cache is true, so on every typing after delay(is set to 500 milliseconds) external service call will be sent (if typed query text was not cached to caching map) to take result and cache it
  • Filtering property (displayField) is: name
  • Autocomplete property useChips is set to true(so autocomplete automatically goes into multiselection state) and selected items will be displayed as chips
  • Created two templates: for dropdown list items and for selected items (chips)
