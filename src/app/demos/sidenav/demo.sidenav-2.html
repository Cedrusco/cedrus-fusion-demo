Demo features:

  • Sidenav is in fullHeight state according to it parent container
  • Sidenav has styling model
  • Content of sidenav section is dynamically changed with CfComponentTemplateService service connected to the Toggle content button
  • Second dynamic content buttons are opening dialogs with different options. Dialogs are also using CfComponentTemplateService service.
