Demo features:

  • To open dialog aligned to some clicked button is needed to set clicked event object to sourceEvent parameter of dialog options. Also together with sourceEvent you may use position parameter to align dialog window to your preference (select by radio).
  • To open dialog with it overlay container inside some component area is needed such steps:
    • make component to extend CfCoreComponent (see how ElementRef and TemplateService dependencies are included inside demo component constructor)
    • in dilalog options set targetId parameter with value this.id (which is provided by CfCoreComponent)
    • put [id]="id" to container inside which you want to open the dialog
